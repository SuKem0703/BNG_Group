# 🎮 Welcome to Chronicles of Knight and Mage

Welcome to **Chronicles of Knight and Mage**, a 2D pixel adventure game developed with Unity.

## 🌍 About the Game
Chronicles of Knight and Mage is set in a mysterious world where magic and darkness collide.
Once a land of harmony, Elaria was shattered by a catastrophic event known as “The Shattering Day.”

Players take on the role of a wandering knight or an exiled mage, exploring a broken world, battling ancient creatures, and uncovering the truth behind its fall.

## ✨ Features
- ⚔️ Real-time combat system
- 🧙 Character progression (skills, stats, equipment)
- 🎒 Inventory and item system
- 📜 Quest and story-driven gameplay
- 🌎 Multiple maps and environments

## 🛠️ Technologies Used
- Unity Engine
- C#
- JSON (Save/Load system)

## 🚀 Development Status
This project is currently in active development by a solo developer.

Core systems such as movement, combat, and data persistence have been implemented.
Additional features, content, and polish are in progress.

## 🗓️ Release Plan
The game is planned to be completed and submitted to Steam by the end of 2026.
